Privacy

A simple note on data, cookies, and external services

This is a personal academic website. It is designed to be lightweight and does not use analytics, advertising trackers, or marketing cookies.

A cookie banner is not shown because the site does not intentionally set non-essential tracking cookies. If this changes in the future, this page and the consent experience will be updated.

Comments

The News page can load public comments through giscus, a third-party service connected to GitHub Discussions. Comments are not loaded automatically: you need to click the comment button first.

giscus states that it does not use tracking or advertising. If you sign in to comment, GitHub authentication and GitHub's privacy practices apply.

External links

Links to services such as LinkedIn, Instagram, Spotify, GitHub, university pages, conference websites, and publication records open external websites. Those services may process data according to their own privacy policies.
